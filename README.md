# FraudShield AI 🛡️

FraudShield AI is an AI-powered cybercrime detection platform developed for Smart India Hackathon.

## Features

- Detects Scam or Safe messages
- Identifies fraud type
- Calculates risk score
- Provides confidence percentage
- Gives safety advice
- Emergency response steps
- Stores message history
- Modern chat interface

## Fraud Types Supported

- OTP Fraud
- Lottery Scam
- Investment Scam
- Loan Scam
- Job Scam
- Phishing Link
- KYC Update Scam

## Technologies Used

- Python
- Flask
- Scikit-Learn
- HTML
- CSS
- JavaScript
- SQLite
